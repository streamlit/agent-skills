# Streamlit Agent Skills

A collection of agent skills for building production-grade Streamlit apps. Each skill focuses on a specific aspect of Streamlit development, from performance optimization to visual design.

## Skills

| Skill | Description |
|-------|-------------|
| [building-streamlit-chat-ui](building-streamlit-chat-ui/) | Chat interfaces, chatbots, AI assistants |
| [building-streamlit-dashboards](building-streamlit-dashboards/) | KPI cards, metrics, dashboard layouts |
| [building-streamlit-multipage-apps](building-streamlit-multipage-apps/) | Multi-page app structure and navigation |
| [choosing-streamlit-selection-widgets](choosing-streamlit-selection-widgets/) | Choosing the right selection widget |
| [connecting-streamlit-to-snowflake](connecting-streamlit-to-snowflake/) | Connecting to Snowflake with st.connection |
| [customizing-streamlit-theme](customizing-streamlit-theme/) | Custom colors via config.toml, avoiding CSS |
| [displaying-streamlit-data](displaying-streamlit-data/) | Charts, dataframes, column configuration |
| [improving-streamlit-design](improving-streamlit-design/) | Icons, badges, spacing, text styling |
| [optimizing-streamlit-performance](optimizing-streamlit-performance/) | Caching, fragments, static vs dynamic widgets |
| [organizing-streamlit-code](organizing-streamlit-code/) | Separating UI from business logic, modules |
| [setting-up-streamlit-environment](setting-up-streamlit-environment/) | Python environment setup |
| [using-streamlit-custom-components](using-streamlit-custom-components/) | Third-party custom components from the community |
| [using-streamlit-layouts](using-streamlit-layouts/) | Sidebar, columns, containers, dialogs |

## How to Route

**Performance issues or slow apps?** → `optimizing-streamlit-performance`

**Building a new UI?** → Start with `choosing-streamlit-selection-widgets` + `using-streamlit-layouts`

**Building a dashboard?** → `building-streamlit-dashboards` + `displaying-streamlit-data`

**Making it look good?** → `improving-streamlit-design` + `choosing-streamlit-selection-widgets` + `displaying-streamlit-data` + `using-streamlit-layouts`

**Multi-page architecture?** → `building-streamlit-multipage-apps`

**Customizing appearance?** → `customizing-streamlit-theme`

**Setting up a project?** → `setting-up-streamlit-environment`

**App getting complex?** → `organizing-streamlit-code`

**Connecting to Snowflake?** → `connecting-streamlit-to-snowflake`

**Building a chatbot or AI assistant?** → `building-streamlit-chat-ui`

**Need features not in core Streamlit?** → `using-streamlit-custom-components`
